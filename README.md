# Data_Cleaning
Data Cleaning project using Python and Pandas – handling missing values, inconsistent date formats, duplicates, and performing data preprocessing for analysis.
This project focuses on cleaning and preparing raw data for analysis using Python (Pandas, NumPy).

-  Tasks Performed:

Handling missing values (NaN)

Replacing incorrect values (e.g., 0 in Age column)

Cleaning inconsistent date formats

Converting columns to proper data types

Removing duplicates

Filling categorical missing values using mode

Validating unique IDs

Basic exploratory data analysis (EDA)

-  Key Concepts Used:

pd.to_datetime()

pd.to_numeric()

fillna()

groupby()

Boolean indexing

Lambda functions

Conditional updating using .loc[]
